# Unique ID Suffix Added to File Names

## Summary

Added timestamp-based unique IDs to all handoff file names to prevent conflicts when multiple agents write files simultaneously.

## Changes Made

### Pattern

All file names now use `$(date +%s%N)` to generate a nanosecond-precision timestamp:

```bash
MSG_FILE=".chatroom/tmp/handoff/message-$(date +%s%N).md"
echo "content" > "$MSG_FILE"
chatroom handoff <id> --message-file="$MSG_FILE" --next-role=...
```
